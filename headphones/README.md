This project involves developing a responsive landing page based on a Figma design file. The primary goal is to translate the visual specifications into functional HTML and CSS, adhering strictly to the required layout, typography, and interactive behaviors.

## Project Requirements and Technical Specifications

### üìê Layout and Responsiveness

| Requirement | Specification | Implementation Note |
| :--- | :--- | :--- |
| **Max Content Width** | **1000px** | All main content must be contained within 1000px and horizontally centered on the page. |
| **Mobile Breakpoint** | Screen width of **480px or less** | Use CSS media queries to implement the mobile-specific design layout for screens that are 480px wide or smaller. |
| **Figma Values** | Float Values | Floating-point values encountered in the Figma file (e.g., spacing, font sizes) should be rounded to the nearest integer for implementation. |

### üé® Styling and Typography

| Feature | Value / Action |
| :--- | :--- |
| **Primary Fonts** | `Source Sans Pro` and `Spin-Cycle-OT` | These fonts must be used to ensure visual fidelity with the design. |
| **Links (`<a>` tags)** | Hover/Active Color: **`#FF6565`** | Apply this color to links when the user hovers over or activates them. |
| **Buttons (`<button>` tags)** | Hover/Active Opacity: **`0.9`** | Apply an opacity change to buttons when the user hovers over or activates them. |
